ReqFlow AI | Manual de Usuario

Version 1.0 · 5 de junio de 2026 · Equipo Grupo 1

ReqFlow AI es una herramienta web para crear, generar, revisar y aprobar requisitos de software asistidos por IA. Integra flujo analista-revisor, feedback de LLM y exportacion HTML. Mantiene historial de versiones para trazabilidad de punta a punta.

Guia de Inicio Rapido

1

Abrir http://localhost:3000.

2

Ingresar con analista@tcs.com / analista123.

3

Crear un nuevo requisito desde el dashboard.

4

Hacer clic en Generar con IA.

5

Enviar a revision para ejecutar el revisor IA.

6

El revisor humano aprueba o devuelve con feedback.

Descripcion de Pantallas

Login

Formulario de acceso con email y password. Disponible para analista y revisor.

Dashboard

Muestra metricas, filtros y tarjetas de requisitos. El analista ve boton para crear.

Nuevo Requisito

Permite seleccionar proyecto, ingresar titulo y descripcion detallada.

Detalle de Requisito

Panel izquierdo con metadatos y acciones. Panel derecho con especificacion generada.

Pantalla de Revision

El revisor compara requisito, especificacion y evaluacion IA antes de aprobar o rechazar.

Historial de Versiones

Timeline con versiones, modelo utilizado, previews, puntaje IA y decision humana.

Guia por Rol

Analista de Negocio

Inicio de sesion ↓ Dashboard → Ver mis requisitos ↓ Nuevo Requisito → Ingresar titulo + descripcion ↓ Detalle → Generar con IA → Revisar spec → Enviar a Revision ↓ Esperar aprobacion ↓ Si rechazado: Re-generar con feedback Si aprobado: Exportar especificacion

Revisor

Inicio de sesion ↓ Dashboard → Ver requisitos En Revision ↓ Abrir Revision → Ver spec + feedback IA ↓ Agregar comentarios propios ↓ Aprobar → requisito aprobado o Rechazar → analista recibe feedback

FAQ

¿Que pasa si la IA tarda mucho? Puede tomar hasta 30 segundos y el backend reintenta automaticamente.

¿Puedo editar la spec generada? Si, mientras el requisito este en borrador.

¿Que significa el puntaje del Revisor IA? Es una evaluacion 0-100; sobre 70 queda pre-aprobado.

¿Se pierden versiones anteriores? No, el historial conserva las versiones.

¿Puedo exportar antes de aprobacion? No, solo especificaciones aprobadas.

¿Que modelos LLM usa? GPT-4o para generar y GPT-4o-mini para revisar.

¿Como incorporo feedback? El sistema lo usa como contexto en la re-generacion.

¿Puedo cambiar de rol? No, el rol lo asigna administracion.
